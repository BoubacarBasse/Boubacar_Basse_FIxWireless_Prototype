# Fix Wireless Operations Dashboard
**Candidate:** Boubacar Basse | **Role:** AI Business Analyst — ServiceCentral

## How to run
1. Download `fixwireless_dashboard.html`
2. Open it in Chrome
3. No installation, no build step, no server needed

## What's inside
- **Parts Inventory tab** — cross-location stock alerts, imbalance scores, sorted by severity
- **Ticket Aging tab** — open tickets by age bucket (green/amber/red), region filter, worst locations first
- **Notifications tab** — SMS template preview and setup guide for franchise owners

## Technical details
- Built with: HTML, CSS, vanilla JavaScript
- API: RepairQ sandbox (mock data fallback included for CORS environments)
- AI tool used: Claude (claude.ai)
